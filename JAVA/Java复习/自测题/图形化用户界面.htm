什么是事件、事件源和事件监听器？简述使用事件监听器进行事件处理的过程。

事件：对界面的某种操作，如按键、单击鼠标、鼠标移动等。

事件源：事件的产生者，通常是某一组件，如按钮等。

事件监听器：是某些事件的接口，其中定义了被监听的事件发生时系统要调用的方法。

事件处理过程由下面3步组成：

①为类声明需要的事件监听器接口。

②给事件源注册一个事件监听器。

③实现接口声明中事件处理方法。
