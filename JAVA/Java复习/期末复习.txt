第6、7章 类的继承、多态
1.继承方式，子类、父类构造方法的调用顺序，容易出错的地方
	子类构造方法中没有显式调用父类的构造方法，则默认调用父类无参的构造方法。
	父类有含参构造方法时，必须提供不含参的构造方法。
2.super关键字的使用方法，与this关键字的区别，两个关键字不能并存
3.重写的特点，与重载方法的区别？
	重写三同：返回类型、方法名、参数列表
	重写2要求：final方法不能重写，不缩小访问权限
	重写1限制：不能重写不被继承的方法（private方法）
4.抽象类、抽象方法之间的关联关系
	1）抽象方法不含方法体，必须显式定义为abstract，否则必须提供方法体，即便方法体为空
	2）有abstract的方法的类必须定义为abstract
	3）抽象类中可以没有抽象方法
	4）子类未全部实现父类的所有抽象方法，该子类也必须定义成abstract
	有抽象方法的类一定是抽象类，不含抽象方法的类不一定不是抽象类
5.接口的特点
	包含常量和方法的声明，不含变量和方法的实现。
	默认权限是public，即便没写任何权限修饰符（不可设置低于public权限的修饰符）
	常量必须初始化，即便没有final关键字
	方法可加可不加abstract/public关键字，不能有方法体，即便方法体为空也不行（即不能有{}）
	接口不能继承类
6.接口和类的继承书写方式
	class A extends B(类名) implements C（接口）, D,...{
7. Object类的equals方法
	比较两个对象地址是否相同（重写equals方法的除外，如String类）
	特性：自反性、传递性、一致性
	非空类对象.equals（null)===false
8. 多态的实现机制
	方法的重载（静态多态）
	方法重写+向上转型
	注：属性不具有多态性，多态是一种行为，只有方法才有多态性
9.向上转型和向下转型 
	向上转型：父类的引用指向子类的对象 （父对象 = 子对象）
	向下转型：子类引用指向父类的对象（不推荐，通常会出现编译或允许错误）
10. 多态的表现形式
只有成员方法可表现为多态，静态方法、成员属性不具有多态性。

第8章 包装器类
1. 包装器类重载方法的调用顺序
直接匹配--> 加宽匹配--> 装箱匹配 --> 装箱再加宽匹配 -->不定长匹配
注：装箱后再加宽变成的是Object类型
class H {
	void go(int x, int y) {
		System.out.println("int,int");
	}

	void go(short... x) {
		System.out.println("short…");
	}

	void go(Short x, Short y) {
		System.out.println("Short,Short");
	}
	
	void go(Integer x, Integer y) {
		System.out.println("Integer,Integer");
	}
	
	void go(Object x, Object y) {
		System.out.println("Object,Object");
	}

	public static void main(String[] s) {
		H test = new H();
		short s1 = 5;
		short s2 = 6;
		test.go(s1, s2);
	}
}
